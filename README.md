# 簡樸粵拼 Plain Jyutping
macOS 簡樸粵拼輸入源

## 點樣將佢添加到輸入法
下載之後雙擊 `PlainJyutping.inputplugin` ，系統就會將佢添加到輸入源列表。
> 雙擊之後，系統會將佢複製到 `~/Library/Input Methods` ，再將佢添加到輸入列表。  
> 如果你雙擊之後見毋到有粵拼，可以手動操作邇個流程。

## 使用
打字箇陣，打完音節之後要撳一下空格掣
